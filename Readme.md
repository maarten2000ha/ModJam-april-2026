# Game jam april 2026

This game jam will be turned into a mod jam to learn how to create a mod for games.
The game in choice will be Hytale, a sandbox game with out-of-the-box modding support.
This choice was made due to wanting to make the first mod a bit on easy mode, yet be able to use new technologies,
compared to Minecraft which has multiple mod-loaders and frameworks. It's also to exercise in Java and to get more
comfortable with it.

# Goals

- Able to create own NPC
- Able to create roles for the NPC (blacksmith, forester, builder, cook...)
- Able to create rooms like in dragon quest builder 2 (blacksmith, forester hut, builders hut, restaurant...)
- Have tests written for it

# Objective

- Able to create own NPC
  - Create custom NPC (Primary) - DONE
  - Able to spawn them in the world (automagically)
  - Custom model and texture
  - Custom animations
- Able to create roles for NPC's
  - Blacksmith (Primary)
    - Able to smelt ore's
    - Able to craft armor
    - Able to craft weapons
  - Forester (Primary)
    - Able to chop wood
    - Able to plant trees
  - Builder 
    - Able to build structures
    - Able to use the builder's workbench
    - Able to repair structures at were build earlier by the builder
  - Cook
    - Able to cook items
  - Doctor
    - Able to heal other NPC's
  - Courier
    - Able to transfer items from the huts to the warehouse and back
  - Crafter
    - Able to craft furniture
    - Able to craft tools from the workbench
- Able to create rooms like in DQB2
  - Blacksmith's hut (Primary)
    - Blacksmith's anvil
    - Armorer's workbench
    - Furnace
  - Forester's hut (Primary)
    - Woodcutter's block
    - Farmer's workbench
  - Builder's hut
    - Builder's workbench
  - Kitchen
    - Chef's stove
    - Furnace
  - Doctor's office
    - Bed
  - Bedroom
    - Bed
  - Warehouse
    - Chests
- Have test written for it
  - Unit tests written
  - Integration test written
  - Research how to write game tests
